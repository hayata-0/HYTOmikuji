# HYTOmikuji
* このライブラリは技育祭2022春の勉強会で行われたハンズオンで勉強した内容になります。

### Installation
Using Xcode 11, go to File -> Swift Packages -> Add Package Dependency and enter `https://github.com/hayata-0/HYTOmikuji`

* import
```html
import HYTOmikuji
```
